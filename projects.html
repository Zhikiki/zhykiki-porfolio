MEET APP

Meet app is a progressive web app (PWA) that lets you search for tech events in your city. App is using the Google Calendar API and Google OAuth2 for authentication. Meet app is built as serverless project with the help of Amazon Web Services (AWS), using a test-driven development (TDD) technique. Techstack for running tests includes Jest, Puppeteer, Atatus. Recharts library is used for data visualisation.

POKEDOX APP

Simple web application with HTML, CSS, and JavaScript that fetches data from an external API, populate it and displays in DOM (Pokemons cards). Each individual card contains a button, which brings the user to modal with detailed information. Navbar has live search/filter. Bootstrap 4.3.1 is used for the user interface elements.
