myFlix Server Side API

The server-side component of myFlix app. This app utilizes a RESTful API to store and display movie and user information using the MERN stack. Express was used routing. App uses MongoDB noSQL database deployed on MongoDB Atlas. The API is tested in Postman. Basic HTTP auth is used for first login then JWT authentication for further API calls. Project is deployed with Herku.

myFlix full-stack project

Case Study

Introduction

myFlix is a web application developed with the MERN stack. It provides users with access to information about different movies, including information about the film’s director as well as which genre specific movie belongs to. Users can sign up, update their personal information, create a list of their favorite movies. Project’s back-end part also provide the possibility to implement the ability to add new movies to the movie library and update movie’s information.

Live Demo

Purpose

This project was built as part of my web development course at CareerFoundry to demonstrate mastery of full stack JavaScript development.

Objective

The aim was to create ambitious full-stuck project by building complete server-side and client-side from scratch to add to my professional portfolio.
